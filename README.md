# 🏥 HMS — Hospital Management System

A full-stack **Hospital Management System (HMS)** designed to manage hospital operations like patients, doctors, and appointments efficiently.

---

## 🚀 Live Demo

👉 [https://hms-theta-mauve.vercel.app/](https://hms-theta-mauve.vercel.app/)

---

## 📌 Overview

The Hospital Management System (HMS) is a web application that simplifies hospital workflows by providing a centralized platform to manage:

* Patient records
* Doctor details
* Appointments
* Administrative tasks

---

## ✨ Features

### 👨‍⚕️ Patient Management

* Add, edit, and delete patient details
* Maintain medical history
* Easy search and filtering

### 🩺 Doctor Management

* Manage doctor profiles
* Assign specialization
* Track availability

### 📅 Appointment System

* Schedule appointments
* Manage booking details
* Prevent conflicts

### 📊 Dashboard

* Overview of system data
* Total patients, doctors, and appointments

### 🔐 Authentication *(if implemented)*

* Secure login/signup
* Role-based access control
